# Genius Lyrics Copier
Instantly copy song lyrics from Genius.com to your clipboard with a single click on the extension icon. No extra steps.

## Options
Customize which characters are replaced during copying by right-clicking the extension and selecting **Options**.
- **Quotes**: Replace smart quotes ( `‘ ’ ‚ “ ” « »` ) with plain quotes ( `' "` ). (default: **on**)
- **Dashes**: Replace various dashes ( `— –` ) with standard hyphens ( `-` ). (default: **off**)
- **Spaces**: Replaces special spaces (non-breaking, etc.) with standard spaces. (default: **on**)
- **Other Characters**: Replaces other characters like ellipsis ( `…` ) and cyrillic 'e'. (default: **on**)

Not affiliated with Genius.com.
